# Contributing to Questarr

Thank you for your interest in contributing to Questarr! We welcome contributions from everyone.

## Getting Started

1.  **Fork the repository** and clone it to your local machine.
2.  **Install dependencies**: `npm install`.
3.  **Start the development server**: `npm run dev`.
4.  **Run tests**: `npm test`.

## Development Guidelines

-   **Code Style**: We use Prettier and ESLint. Run `npm run format` and `npm run lint` before committing.
-   **Testing**: Add unit tests for new features and bug fixes. Ensure all tests pass before submitting a PR.
-   **Commits**: Use descriptive commit messages.

## Pull Requests

-   Open a pull request against the `main` branch.
-   Follow the PR template.
-   Ensure CI checks pass.
